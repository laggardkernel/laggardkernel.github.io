Downgrade the Kernel

小心，需要同时降级

  • linux
  • linux-headers
  • kernel modules like virtualbox-host-modules-arch-5.2.8-4-x86_64.pkg.tar.xz

Find theme in /var/cache/pacman/pkg

Arch Linux Archive

ALA所能保存的历史版本有限。

ALA stores official repositories snapshots, iso images and bootstrap tarballs across time.

You can use it to

  • Downgrade to a previous version of one package (last version is broken, I want the previous one)
  • Restore all your packages at a precise moment (my system is broken, I want to go back 2 months ago)
  • Find a previous version of an ISO image

https://archive.archlinux.org/
