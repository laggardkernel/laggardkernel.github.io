强制 MacBook Touchbar显示为 Function Key 功能键

2018-02-09
gui
830 words 2 mins read

Contents

  • TouchBar 隐藏设置项
  • defaults 命令设置 Touch Bar 默认显示为功能键 F1-F12
  • 检查应用对应的 bundle id

Touchbar 上不能默认显示功能键，苹果只给出单独设置某个应用默认为功能键的方案，参考如何在配备 Multi-Touch Bar 的 MacBook Pro 上使用功能键。但实际上存在隐藏配置项，可以将功能键作为默认显示。

更新：貌似 macOS 10.13 将此隐藏项放到了系统设置中。也就是说，此项不再隐藏。本文唯一意义是实现此更改的完全命令行操作。

TouchBar 隐藏设置项

思路来源自此 reddit 提问，提问者需要添加 JDK 为默认显示功能键，有回答者给出了 defaults 命令修改 com.apple.touchbar.agent domain，直接添加 JDK 绝对位置。（因为 JDK 没有安装到 /Applications 下，“键盘设置–快捷键–功能键”无法添加此应用。

首先在 “System Settings - Keyboard” 更改 “Touch Bar show”, “Press Fn key to”，以及 “System Settings - Keyboard - Shortcuts” 中更改 “Function Keys” 项。这样的话，com.apple.touchbar.agent 域中值将发生变化。

❯ defaults read com.apple.touchbar.agent
{
    PresentationModeFnModes =     {
        appWithControlStrip = functionKeys;
    };
    PresentationModeGlobal = fullControlStrip;
    PresentationModePerApp =     {
        "com.binarynights.ForkLift-3" = functionKeys;
        "com.microsoft.VSCode" = functionKeys;
        "com.sublimetext.3" = functionKeys;
    };
}

通过输出 com.apple.touchbar.agent 中值，可以很容易推测出它们与 “System Settings” 设置的对应关系。

System Settings - Keyboard com.apple.touchbar.agent Available values
Touch Bar show PresentationModeGlobal appWithControlStrip(default), app, fullControlStrip, functionKeys
Press Fn key to PresentationModeFnModes same with above.
Shortcuts - Function Keys PresentationModePerApp same with above.

defaults 命令设置 Touch Bar 默认显示为功能键 F1-F12

问题就在于 “Touch Bar show” 没有给出对应 functionKeys 的下拉选项。所以，直接通过命令行修改接口达成目的。

# 默认显示功能键
defaults write com.apple.touchbar.agent PresentationModeGlobal functionKeys \
  && defaults write com.apple.touchbar.agent PresentationModeFnModes {}

# 默认显示功能键，Fn 按下后依然显示功能键
defaults write com.apple.touchbar.agent PresentationModeGlobal functionKeys \
  && defaults write com.apple.touchbar.agent PresentationModeFnModes {} \
  && defaults write com.apple.touchbar.agent PresentationModeFnModes -dict-add functionKeys functionKeys

# 重启TouchBarServer，使其重新读取配置
sudo killall TouchBarServer

另外，直接注意的是，PresentationModePerApp 没有其对应项，也就是目前没有为某个应用单独设置Fn按下时 TouchBar 显示内容。Fn按下时 TouchBar 显示内容只存在一个全局设置项。
