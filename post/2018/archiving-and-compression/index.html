在命令行里该用什么命令解压什么文件？压缩命令使用总结。个人笔记，非入门教程。

压缩格式不重要，关键是其使用了什么压缩算法。

工具选择建议

  • 分享Linux文件最好使用 tar 包，可以保留文件权限（默认不拷贝 attrs, xattr）
  • p7zip 用来解压各种格式，以及压缩 7z 格式。支持解压 rar 压缩包。
    • 可替代 unzip，unrar
  • zip encoding，由于压缩包没有记录编码，需要工具支持自定编码参数
    • unzip-iconv，-O, -I. Ubuntu 已经使用了此patch. macOS 直接使用 unar
    • unarchiver on Linux, unar on macOS, urar -e gb18030. (此包只支持解压)
    • iconv and convmv
  • 配合 atool 避免记忆压缩、解压参数
  • pv for progress in case you use custom compresion options in tar

gzip, bzip2, xz

gzip 支持解开 compress，zip，gzip 压缩文件

gzip 命令压缩、解压后默认删除原来文件

zcat/zmore/zless/zgrep 一系列工具

bzip2 默认不会删除原来文件需要显式使用 -k 参数。

tarball

tar 压缩，-z gzip, -j bzip2, -J xz

  • -p，保留权限与属性
  • -P，保留绝对路径。默认移除开头/
  • 解压部分文件 tar -jxv -f /root/etc.tar.bz2 etc/shadow

bsdtar v.s. gnutar

While BSD tar recognizes compression formats based on the format, GNU tar only guesses based on the file extension.

BSD tar包名为 libarchive。命令名为 bsdtar。

压缩算法

注意压缩算法。xz GNU实现的LZMA压缩。

p7zip 实现了多种压缩，LZMA2（加入多线程），不支持保存Linux文件权限。

  • gzip, Burrows–Wheeler algorithm
  • bzip2, DEFLATE
  • xz, LZMA
  • 7zip, LZMA2, etc

Fuck rar

unrar is rar official app, not open sourced.

unar is command from unarchiver pkg, with support for more than 40 archive formats. 仅支持解压和查看。

1

并行压缩

GNU 实现的gzip，bzip2，xz均不支持并行。可以使用对应并行包进行压缩，解压不受影响。（参考archlinux wiki查看对应并行方案）

  • pigz replaces gzip
  • libzip2 or pbzip2 replaces bzip2
  • pixz, pxz replaces xz
